# HabitLink 任务分工与 10 天计划

## 四人分工

| 成员 | 角色 | 主要职责 | 交付物 |
| --- | --- | --- | --- |
| 成员 A | 项目负责人 / 后端 | 需求拆分、数据库设计、用户与登录接口、接口联调 | 需求文档、数据库脚本、用户模块接口 |
| 成员 B | 后端开发 | 目标模块、打卡模块、防重复打卡、统计逻辑 | 目标接口、打卡接口、统计接口 |
| 成员 C | 前端开发 | Vue3 项目搭建、登录注册页、目标列表、打卡页面 | 前端页面、Axios 封装、路由配置 |
| 成员 D | 前端与测试 / 答辩 | 小组页面、接口测试、演示数据、PPT 和答辩材料 | 小组页面、测试记录、答辩 PPT |

## 10 天计划

| 天数 | 计划内容 | 目标 |
| --- | --- | --- |
| 第 1 天 | 明确需求、确定 MVP 范围、建立项目目录 | 完成需求文档和基础结构 |
| 第 2 天 | 设计数据库表结构和接口文档 | 完成 `init.sql` 与 `api.md` |
| 第 3 天 | 搭建 Spring Boot 后端基础工程 | 完成后端项目骨架、数据库连接 |
| 第 4 天 | 实现用户注册登录接口 | 用户模块可通过接口测试 |
| 第 5 天 | 实现目标创建、目标列表和目标详情接口 | 目标模块可用 |
| 第 6 天 | 实现今日打卡、防重复打卡和记录查询 | 打卡闭环可用 |
| 第 7 天 | 实现累计天数、连续天数统计 | 统计功能可展示 |
| 第 8 天 | 实现小组创建、邀请码加入、成员列表 | 小组功能可用 |
| 第 9 天 | 实现前端页面联调，补充演示数据 | 完成 MVP 主流程演示 |
| 第 10 天 | 测试、修复问题、整理 README 和答辩材料 | 完成课程提交版本 |

## 阶段验收标准

- 用户可以注册登录并进入系统
- 用户可以创建目标并完成今日打卡
- 同一用户同一目标同一天不能重复打卡
- 用户可以查看打卡记录、累计天数和连续天数
- 用户可以创建小组并通过邀请码加入
- 小组页面可以查看成员及今日打卡状态

## MVP 之外的预留任务

- 密码加密存储和登录 Token 过期机制
- 统一异常处理和参数校验
- 前端路由守卫和登录态持久化
- 部署脚本、生产环境配置和 HTTPS
- 微信小程序端登录、头像同步和移动端页面适配
